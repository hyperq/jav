# JAV TUI - 完整功能规划

## 数据模型

### 网站 URL 结构

| 功能 | URL | 说明 |
|------|-----|------|
| 有码首页 | `/` | 列表页, `.movie-box` |
| 无码首页 | `/uncensored` | |
| 搜索影片 | `/search/{keyword}` | 翻页: `/search/{keyword}/page/{n}` |
| 有码女优列表 | `/actresses` | `.avatar-box` |
| 无码女优列表 | `/uncensored/actresses` | |
| 搜索有码女优 | `/searchstar/{keyword}&type=&parent=ce` | |
| 搜索无码女优 | `/uncensored/searchstar/{keyword}&type=&parent=uc` | |
| 女优作品页 | `/star/{code}` | 跟列表页结构相同 |
| 影片详情页 | `/{number}` (如 `/OFJE-614`) | |
| 磁链 AJAX | `/ajax/uncledatoolsbyajax.php?gid=...` | |

### 数据结构

```
Movie {
    number, title, link, cover, date, tags[]
}

Actress {
    code,        // "okq"
    name,        // "三上悠亜"
    avatar,      // "/pics/actress/okq_a.jpg"
    birthday, age, height, cup, bust, waist, hip, hobby
}

MovieDetail {
    number, title, cover,
    date, duration, maker, publisher,
    genres[], actresses[],
    sample_images[], magnets[]
}

Magnet { link, size, caption }
```

## 四大视图 (Tabs)

### 1. 🔍 影片搜索
- `f` 弹出搜索框, 输入关键词
- 左面板: 卡片列表 (封面+标题+标签+番号)
- 右面板: 选中影片的详情 (大封面+元数据+磁链)

### 2. 👩 女优浏览
- 左面板: 女优卡片列表 (头像+姓名+信息)
- `f` 搜索女优名
- 选中女优 → 右面板切换为该女优的作品列表
- 在作品列表中 Enter → 查看详情/磁链

### 3. 📋 历史
- 左面板: 浏览过的影片记录
- 右面板: 选中条目的详情

### 4. ⭐ 收藏
- 左面板: 已收藏的影片
- 右面板: 选中条目的详情

## 交互流程

```
启动 → 搜索弹窗(f)
         ↓ Enter
      影片列表 ←→ 详情面板
         ↓ Tab切换
      女优列表 → 选中女优 → 右面板变为作品列表
                              ↓ Enter
                           影片详情/磁链
```

## 快捷键

| 键 | 全局 | 说明 |
|----|------|------|
| f, / | ✓ | 弹出搜索 (根据当前tab搜影片或女优) |
| Tab | ✓ | 左右面板切换 |
| ←→ | ✓ | tab 切换 |
| j/k | ✓ | 列表上下导航 |
| Enter | ✓ | 选中/确认 |
| n/p | ✓ | 翻页 |
| y | 详情 | 复制磁链 |
| s | 详情 | 收藏 |
| ~ | ✓ | 显示/隐藏日志面板 |
| q | ✓ | 退出 |

## 技术要点

- 所有 HTTP 请求禁用重定向 (302 body 已含数据)
- 图片请求需 Referer 头
- 缩略图/封面异步加载, Kitty Protocol 渲染
- 磁链通过 AJAX 接口获取 (需解析页面 JS 变量 gid/uc/img)
- SQLite 存储历史和收藏
